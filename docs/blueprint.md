# **App Name**: Danesh Shekh Lane Ram Nabami Udjapan Committee

## Core Features:

- Bilingual Content Engine: Seamless English and Bengali language switching throughout the app using an i18n implementation.
- Real-time Live Counters: Dynamic animated counters and event countdown timers powered by real-time Firestore snapshots.
- Member Directory & Search: A searchable, filtered database of committee members with CSV upload capabilities and pagination managed via Firestore.
- Dynamic Admin CMS: A comprehensive control panel to manage work items, news headlines, and upcoming events without modifying code.
- Media Showcase Hub: Integrated photo and video gallery utilizing Firebase Storage for community work updates and local events.
- AI Bilingual Post Enhancer: An AI-powered tool to refine or translate event descriptions and news snippets between Bengali and English.
- Social News Feed: Cards for community news that include social share triggers and auto-playing video reel previews.

## Style Guidelines:

- Primary Color: Rich Heritage Amber (#CC7000), a sophisticated, saturated hue evoking tradition and community warmth.
- Background: Deep Charcoal Basalt (#14110F), providing a high-end, premium dark mode aesthetic for superior legibility.
- Accent: Polished Gold (#EBC033), used sparingly for CTAs, counters, and highlights to signify prestige.
- Headline font: 'Playfair' (serif) for formal elegance; Body font: 'PT Sans' (sans-serif) for modern humanist legibility across languages.
- High-impact editorial layout featuring large hero images, generous negative space, and full-bleed visual sections.
- Orchestrated page transitions and staggered scroll-reveal animations for card elements to maintain a fluid, living website feel.
- Custom minimal line-art icons that feel artisanal, ensuring visual consistency across both English and Bengali interfaces.
